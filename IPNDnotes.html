The Basics of the Web and HTML

Overview of the Internet

The Web is a network of computers. There are at least 4 concepts that are necessary to understand how it works:

  1. Browsers
  2. The Internet
  3. HTTP
  4. Servers

Put in simple terms, users interface themselves with the internet though a web browser. From there they are able to send HTTP protocol requests that will travel through the internet to reach a server where the requested document is stored.

The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.

What is a Web Page

A web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.

Basic HTML Vocabulary that will be useful to remember:

  1. Tag An HTML tag is always contained within angled brackets. Most tags have an opening tag (<p> for example) and a closing tag, (</p>). Some tags (called "void" tags) do not require a closing tag (like the <br> tag).
  2. Element An HTML element refers to everything within a set of opening and closing tags.
  3. Attribute This is a property of an HTML element. For example, to set the hrefattribute of an anchor tag to the Udacity URL, you would write <a href="www.udacity.com">
What is HTML

HMTL stands for Hypertext Markup Language. HTML documents form the majority of the content on the web and usually contain Text and Markups as well as references to other documents and links to other pages.

Creating a Structured Document

HTML, CSS and Javascript

Html, CSS and JS are the three languages that web development is based on, each one has different purposes, but in this course we’ll cover only Html and CSS.

Html and CSS change over time, but learning their basic principles will allow us to learn new techniques much faster as they tend not to change over time.

HTML files are the structure of a webpage, imagine the walls of our home. CSS files define the Style, or what color the walls are in our analogy. Javascript files are the interactive components.

Developer Tools

Some browser, specifically Chrome, have some built in futures called Developer Tools that allow us to see, and modify if we want, the structure of a webpage in Html and CSS code.

We can also inspect whatever element we want on the website, and the browser will highlight which code generates that element in the developer tool.

DOM

DOM or Document Object Model, are the tree structure of a webpage or how the browser interprets code to actually create the webpage structure.

To render a document such as an HTML page, most web browsers use an internal model similar to the DOM. The nodes of every document are organized in a tree structure, called the DOM tree.

HTML being the language of the web, has its specific syntax and rules, the basic word in the html language is a tag, which the browser turns into elements that then form the DOM tree.

A tag can also have attributes with values.

CSS has also syntax and rules, but does not impact the structure of the website but the rules, for example we can specify that every element belonging to a specific class ‘’x’’ will have a specific color and so on.

Boxes

The structure of a website is entirely composed of boxes. This is easy to identify when playing arounf with the developer tool in the browser, highliting an element will highlight the entire box.

No matter what shape it contains, it can be a circle like the example in the lesson, the form of the element and the space it occupies in the webpage is a box or a rectangle.
All of these boxes put together compose the structure of the website.

Adding CSS Style to HTML Structure

How does CSS control HTML?

We can either assign a style through CSS to any HTML element like h1, h2, p etc. which will then be the ''selectors'', and this is fine if we want every one of a certain element to have the same style but if we want to have h1's with different styles we will have to use classes, in which case we will be dealing with class selectors.

How can I include styling to my webpage with CSS?

There are 3 methods for this but I will only note down 2 because the third is messy and complicated, I don't need that.

The first method is to write it directly in the ''head'' of our HTML file by opening a ''style'' tag.

The second method and the one that seems to make more sense to me, also because it's cleaner, is to have a separate CSS file containing the CSS code, which is then linked to our HTML file.

Positioning and resizing boxes

It is good to build a habit of defining sizes in percentages rather than pixels, that way the boxes can resize themselves depending on the situation.

To position them, we have the option of using, display: flex which will override CSS's default behaviour and position the boxes next to each other.

Introduction to Serious Programming

Computer Science and Programming

Computer Science is about how to solve problems, breaking them into smaller pieces and then precisely and mechanically describing the steps that can be used to solve each piece, and those steps can be executed by a computer.

Programming is at the core of computer science, a computer can be programmed to do almost anything and do it extremely fast.

Machines on the other hand, are able to do only one thing, or very few things at the most, but if they're not programmed correctly, computers can be less useful than machines.

Programming Languages

Programming languages is what programmers use to give instructions to the computer by writing programs.

In this course we will focus on one in particular, which is Python, Python is also what is also called a high level language.

Why do we need programming languages?

Considering their purpose programminf languages have several advantages over natural languages, mainly they are better at avoiding these two things:

  1. Ambiguity, the computer needs to interpret the input of the programmer, and interpret them exactly the way the programmer intended. Some words in English for example can have 2 meanings, like the word ‘’biweekly’' this ambiguity can’t be allowed in a programming language.
  2. Verbosity, natural languages are verbose, meaning a lot of text is necessary to describe a complicated thing, not true for programming languages like Python.
Python

We said in the previous paragraph that Python is a high level programming language, but what it really is for the computer is an interpreter, so Python itself is also a program.

The programs that we will write won't run directly on the computer, but be the input to the Python program which then runs on the computer.

Python Expressions

A Python "expression" is a legal Python statement. For example: ''print 1 + 1'' is a valid expression, but ''print 1 + '' is not. The python grammar for arithmetic expressions is explained below:

  1. Expression = Expression-Operator-Expression OR Number
  2. Operator = +,-,*,/ etc...
  3. Number = 0,1,2 etc...

So expressions can be replaced either by a number or another expression, the expression is not done until we've reached all the terminals.

Also worth nothing that Python does something called integer division which means it ignores the decimal part of the answer.

To force Python to give you an answer including a decimal, you need to make one of the numbers into a decimal by putting a decimal point after it.

Variables and Strings

What Is a Variable?

Variables give programmers a way to give names to values.
If my_variable is a variable with a value of 2, then the following code would print out 4:
print my_variable + my_variable

How to assign a value to a variable

We can assign the value 2 to the variable my_variable by writing code like this:
my_variable = 2

We can even change the value of a variable by re-assigning it to a different value later.

The difference between what the equals ''='' means in math versus in programming.

In the line 2 + 3 = 5, the equals sign means "is the same as".

In the line my_variable = 5, the equals sign means "takes the value of".

The difference between Numbers and Strings

In Python, 2 is a number while "2" is a string.

The code 2 + 2 would give 4.

The code "2" + "2" would give "22".
